# Bunny Buffet

Bunny Buffet is a tiny browser game about building the healthiest bunny diet in only six feeds.

## How it works

- Feed the bunny by clicking one of the food buttons.
- Hay counts as a larger value because a healthy rabbit diet should be mostly hay.
- The game ends after exactly 6 feeds.

## Winning diet

- `80-90%` high-quality grass hay
- `10-15%` fresh leafy greens
- `5% or less` high-fiber pellets

Fruit is included as a tempting treat, but it counts toward the fresh food side of the plate and can still throw off your balance if you use too much of it.

## Food values

- Hay Bundle: `6`
- Leafy Greens: `1`
- Fruit Treat: `1`
- Pellet Scoop: `1`

## How to run it

Open `index.html` in a browser.
